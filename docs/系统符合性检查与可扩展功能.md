### 1.1 题目

**基于大语言模型进行实体和领域增强的专利技术匹配系统**

| 序号            | 要求内容                                                                   |
| --------------- | -------------------------------------------------------------------------- |

**匹配流程**：

```
查询文本
    │
    ▼
┌───────────────────┐
│ LLM提取实体/领域   │
└───────────────────┘
    │
    ▼
┌───────────────────┐
│   混合检索        │
│ ├─ Qdrant向量检索 │
│ └─ ES关键词检索   │
└───────────────────┘
    │
    ▼
┌───────────────────┐
│   LLM精排         │
│ 实体/领域匹配度   │
└───────────────────┘
    │
    ▼
┌───────────────────┐
│   返回匹配结果    │
│ 匹配专利+解释    │
└───────────────────┘
```

**相似度计算公式**：

$$
Similarity = \alpha \cdot Sim_{semantic} + \beta \cdot Sim_{entity} + \gamma \cdot Sim_{domain}
$$

其中：

- $Sim_{semantic}$：语义相似度（基于LLM Embedding向量余弦相似度）
- $Sim_{entity}$：实体匹配度（Jaccard相似度）
- $Sim_{domain}$：领域匹配度（IPC分类距离）
- $\alpha, \beta, \gamma$：权重参数（可配置）

**符合度：100%** ✅
