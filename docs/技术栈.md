✅ 最终完整技术栈（更新版）

1. 基础设施与部署
   类别	技术	说明
   容器化	Docker	打包所有服务
   编排	Docker Compose	一键启停全系统（开发/演示）
   反向代理	Nginx	前端静态资源托管 + 后端 API 路由
2. 大语言模型（LLM）与 AI 能力层
   类别	技术	说明
   本地 LLM 运行时	Ollama	运行 Qwen2、Llama3 等开源模型（ollama run qwen2:7b-instruct）
   LLM 集成框架	Spring AI	统一调用 Ollama 的 Chat 与 Embedding 能力
   向量化模型	BGE / text2vec / Ollama embeddings	通过 EmbeddingModel 生成专利文本向量
   提示工程	Structured Prompt + JSON Mode	引导 LLM 输出结构化实体与 CPC 领域
3. 数据存储层
   类别	技术	作用
   向量数据库	Qdrant	存储专利 embedding + payload（如 {"domain": "G06F", "entities": ["CNN"]}），支持高效语义检索与过滤
   全文检索引擎	Elasticsearch 8.x + ik 分词器	支持中文关键词检索、布尔查询、高亮片段，与 Qdrant 形成混合检索
   对象存储	MinIO	存储用户上传的专利 PDF/DOC 文件，S3 兼容
   关系型数据库	MySQL 8.0+	存储用户信息、专利元数据（公开号、申请人、状态等）
   数据访问	MyBatis-Plus 3.5+	增强版 MyBatis，提供 ActiveRecord、LambdaQuery、分页插件等便捷功能
   数据库驱动	mysql-connector-j	官方 MySQL JDBC 驱动（runtime scope）
   会话缓存	Redis 7.x	存储 Sa-Token 登录会话，支持集群与持久化
4. 后端服务（Spring Boot 3 生态）
   类别	技术	说明
   核心框架	Spring Boot 3.2+ (Java 17+)	构建 RESTful API
   权限认证	Sa-Token	轻量级登录、鉴权、权限控制（替代 Spring Security）
   向量操作	Spring AI Qdrant Starter	QdrantVectorStore 实现向量存取
   全文检索	Elasticsearch 客户端	操作 ES 索引
   文件操作	MinIO Java SDK	上传/下载专利文件
   异步处理	@Async + TaskExecutor	异步解析 PDF、调用 LLM
   API 文档	Springdoc OpenAPI (Swagger)	自动生成接口文档
   日志	SLF4J + Logback	默认集成
5. 前端（Vue 3 生态）
   类别	技术	说明
   核心框架	Vue 3 (Composition API)	响应式开发
   UI 组件库	Element Plus	Vue 3 官方推荐组件库（替代 ElementUI）
   状态管理	Pinia	轻量、TypeScript 友好
   路由	Vue Router 4	页面导航
   HTTP 客户端	Axios	调用后端 API，拦截器自动携带 Token
   构建工具	Vite	快速开发与构建
   可视化	ECharts	展示技术领域分布、实体词云、匹配结果雷达图、专利关系图
6. 部署策略
   类别	技术	说明
   容器化	Docker	所有服务容器化部署
   编排	Docker Compose	一键启动 / 停止全系统
   前端代理	Nginx	静态资源 + 后端 API 反向代理
