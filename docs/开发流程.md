# 毕设系统开发

题目：

基于大预言模型进行实体和领域增强的专利技术匹配系统

要求：

设计一种基于大预言模型进行实体和领域增强的专利技术匹配系统。

1. 构建大预言模型的运行环境，支持主流大预言模型的访问，涵盖在线和离线两种模式。
2. 设计大预言模型引导指令，用于挖掘专利文本中的实体和层次化领域信息；
3. 基于识别的实体和领域信息来增强大预言模型进行专利技术匹配的能力。
4. 采用B/S模式实现一个基于大预言模型进行实体和领域增强的专利技术匹配系统，支持用户进行专利技术信息的录入、检索和可视化等功能。

5.不返回非必要字段，应通过 _source filtering 控制返回内容
